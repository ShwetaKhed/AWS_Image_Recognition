Description of Assignment

Providing AWS Cloud-based solution that exploits services such as S3, Lambda, API Gateway and database service to build a system for automated object-detection tagging and query handling. We have produced a solution that allows end-users to upload their images into an S3 bucket. Upon uploading of an image to a designated S3 bucket, a lambda function is automatically triggered that uses the Yolo object detection feature and finds the list of objects in the image and stores the list of detected objects along with the URL of the image (S3 URL) in a database. Further, the end-user submits queries to an API endpoint using API Gateway to search tagged images
